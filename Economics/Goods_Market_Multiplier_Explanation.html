Goods Market & Multiplier — Рынок товаров и мультипликатор

Equilibrium in Goods Market / Равновесие на рынке товаров

English: Equilibrium when planned spending (demand) = output (supply). Y = Z (planned aggregate demand).

Русский: Равновесие, когда планируемые расходы = выпуск. Y = Z.

Components of Demand / Компоненты спроса

Z = C + I + G + X - M

C = Consumption, I = Investment, G = Gov't spending, X = Exports, M = Imports

В закрытой экономике: Z = C + I + G

Consumption Function / Функция потребления

C = c₀ + c₁Y

c₀ = autonomous consumption (when Y=0)

c₁ = marginal propensity to consume (MPC) — fraction of extra income spent (0 < c₁ < 1)

The Multiplier / Мультипликатор

An increase in autonomous spending (G, I, or c₀) leads to a LARGER increase in equilibrium Y. Why? More spending → more income → more consumption → more income...

Multiplier = 1 / (1 - c₁) = 1 / (1 - MPC)

If MPC = 0.8, multiplier = 5. ΔG = 10 → ΔY = 50

    ΔY = multiplier × ΔG

Paradox of Thrift / Парадокс бережливости

If everyone saves more (c₁ falls), aggregate demand falls → Y falls → savings may not increase. In recession, more saving can worsen the downturn.

Если все сберегают больше → совокупный спрос падает → ВВП падает.
