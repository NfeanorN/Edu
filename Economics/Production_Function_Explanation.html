Production Function — Производственная функция

What is a Production Function? / Что такое производственная функция?

A production function shows the maximum output (Q) that can be produced from given amounts of inputs (e.g. labour L, capital K) with a given technology. It describes the technical relationship between inputs and output.

Производственная функция показывает максимальный выпуск (Q), который можно получить при заданных объёмах факторов производства (труд L, капитал K) и данной технологии. Это техническая связь «ресурсы → продукция».

Q = F(L, K, …)

Q = output (quantity) / выпуск

L = labour / труд

K = capital / капитал

F = technology / технология

Example: With 2 workers and 1 machine → 10 units per day. With 4 workers and 1 machine → 18 units per day. Here K is fixed (short run); only L changes.

Пример: 2 работника + 1 станок → 10 ед. в день. 4 работника + 1 станок → 18 ед. Капитал фиксирован (краткосрочный период), меняется только труд.

Total Product (TP), Average Product (AP), Marginal Product (MP) / Совокупный, средний и предельный продукт

Term / ПонятиеFormula / ФормулаMeaning / Значение
Total Product (TP)Q = F(L, K)Total output / Совокупный выпуск
Average Product (AP)AP = Q / LOutput per unit of labour / Выпуск на единицу труда
Marginal Product (MP)MP = ΔQ / ΔLExtra output from one more unit of labour / Дополнительный выпуск от ещё одной единицы труда

MP answers: “If we add one more worker, how much does output increase?” AP answers: “On average, how much does each worker produce?”

MP — прирост выпуска от ещё одного работника. AP — средний выпуск на одного работника.

Graphs: TP, AP, and MP / Графики TP, AP и MP

Total Product (TP) / Совокупный продукт

TP curve shows how total output Q changes as we add more of the variable input (e.g. labour L), holding other inputs fixed. It typically rises, then may flatten or even fall if too much labour is added (negative MP).

Кривая TP показывает, как общий выпуск Q меняется с ростом переменного фактора (труда L) при фиксированном капитале.

Average Product (AP) and Marginal Product (MP) / Средний и предельный продукт

When MP > AP, the average is pulled up → AP rises. When MP < AP, the average is pulled down → AP falls. So MP crosses AP at the maximum of AP. MP reaches its maximum at the inflection point of the TP curve (before AP reaches its max).

Когда MP > AP, средний продукт растёт. Когда MP < AP, средний продукт падает. MP пересекает AP в максимуме AP.

Law of Diminishing Returns / Закон убывающей отдачи

If one input (e.g. labour) is increased while at least one other input (e.g. capital) is fixed, the marginal product of the variable input will eventually decrease. Adding more and more workers to a fixed amount of machinery yields smaller and smaller increases in output.

Закон убывающей отдачи: При увеличении одного фактора (труд) при неизменном другом (капитал) предельный продукт переменного фактора в итоге убывает.

Three Stages of Production / Три стадии производства

Stage / СтадияCondition / УсловиеDescription / Описание
Stage IMP > AP, AP risingIncreasing returns; underuse of fixed input / Возрастающая отдача
Stage IIMP < AP, MP > 0Diminishing returns; rational zone / Убывающая отдача, рациональная зона
Stage IIIMP < 0Negative returns; too much variable input / Отрицательная отдача, неэффективно

Firms typically operate in Stage II: adding more labour still increases total output (MP > 0) but each extra worker adds less than the average (MP < AP). In Stage III, adding labour reduces total output (MP < 0), so it is irrational to produce there.

Рациональная зона — Стадия II. В Стадии III предельный продукт отрицателен — добавление труда снижает выпуск.

Link to Costs: MP and Marginal Cost (MC) / Связь с издержками: MP и MC

When MP rises, producing one more unit of output requires less extra labour (and other variable inputs) → MC falls. When MP falls (diminishing returns), each extra unit of output requires more variable input → MC rises. So MC is inversely related to MP.

Когда MP растёт → MC падает. Когда MP падает (убывающая отдача) → MC растёт. MC обратно связан с MP.

MP (labour)                    MC (cost)
  |    /\                        |        /
  |   /  \                       |       /
  |  /    \___                   |      /
  | /                           |     /
  |/                            |____/
  0+----+----+----+ L            0+----+----+----+ Q
  When MP rises → MC falls. When MP falls → MC rises.
