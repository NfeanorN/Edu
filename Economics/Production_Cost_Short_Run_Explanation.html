Production & Cost (Short Run) — Производство и издержки (краткосрочный период)

Short Run vs Long Run / Краткосрочный vs долгосрочный период

Short run: At least one input is FIXED (e.g., factory size, land). Only variable inputs (labor, raw materials) can change.

Краткосрочный период: Хотя бы один фактор ФИКСИРОВАН (завод, земля). Менять можно только переменные (труд, материалы).

Marginal Cost (MC) and Average Costs (ATC, AVC, AFC) / Предельные и средние издержки

Marginal Cost (MC) / Предельные издержки

Marginal Cost is the extra cost of producing one more unit of output. MC = ΔTC / ΔQ (change in total cost divided by change in quantity).

Предельные издержки (MC) — дополнительные издержки на производство ещё одной единицы продукции. MC = ΔTC / ΔQ.

MC = ΔTC / ΔQ (or, for smooth curves: MC = dTC/dQ)

Example: if TC goes from $100 at Q=10 to $115 at Q=11, then MC = (115−100)/(11−10) = $15 per unit.

Пример: если TC при Q=10 равен $100, а при Q=11 — $115, то MC = $15 на единицу.

Average Total Cost (ATC) / Средние общие издержки

Average Total Cost is total cost per unit: ATC = TC / Q. It is the cost per unit of output if we spread all costs (fixed + variable) over the quantity produced.

Средние общие издержки (ATC) — издержки на единицу продукции: ATC = TC / Q.

ATC = TC / Q = (FC + VC) / Q = AFC + AVC

Average Variable Cost (AVC) and Average Fixed Cost (AFC) / AVC и AFC

AVC = VC / Q — variable cost per unit. AFC = FC / Q — fixed cost per unit. As Q increases, AFC always falls (fixed cost is spread over more units). AVC typically falls at first (efficiency), then rises (diminishing returns).

AVC = VC / Q — переменные издержки на единицу. AFC = FC / Q — постоянные на единицу. С ростом Q величина AFC всегда падает; AVC сначала обычно падает, затем растёт.

Relationship: MC and ATC / Связь MC и ATC

  • When MC < ATC, producing one more unit pulls the average down → ATC is falling.
  • When MC > ATC, producing one more unit pulls the average up → ATC is rising.
  • Therefore MC crosses ATC at the minimum of ATC (and similarly MC crosses AVC at the minimum of AVC).

Когда MC < ATC, средние издержки падают. Когда MC > ATC, средние издержки растут. Значит MC пересекает ATC в точке минимума ATC (и AVC — в минимуме AVC).

Graph 2: MC as the slope of Total Cost (TC) / MC как наклон общей суммы издержек

MC = slope of the TC curve at each output level. When TC is steep, MC is high; when TC is flat, MC is low. So MC is the additional cost of one more unit — geometrically, the rise in TC when Q increases by 1.

MC = наклон кривой TC в каждой точке. Крутой подъём TC → высокий MC; пологий TC → низкий MC.

Production Function: Marginal Product / Производственная функция: предельный продукт

Marginal Product (MP) = extra output from one more unit of variable input (e.g., labor).

Предельный продукт = дополнительный выпуск от ещё одной единицы переменного фактора.

Output (Q)     MP
  |             |
  |    /\       |    /\
  |   /  \      |   /  \
  |  /    \___  |  /    \___  (diminishing returns)
  | /          | /           (убывающая отдача)
  |
  0+----+---- L  0+----+---- L

Law of Diminishing Returns: As you add more of a variable input (holding fixed inputs constant), marginal product eventually decreases.

Закон убывающей отдачи: Предельный продукт в итоге падает при добавлении переменного фактора.

Relationship: MP and MC / Связь MP и MC

When MP rises → MC falls. When MP falls → MC rises. MC is inversely related to MP.

Когда MP растёт → MC падает. Когда MP падает → MC растёт.
