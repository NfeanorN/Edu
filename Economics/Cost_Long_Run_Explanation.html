Cost Functions (Long Run) — Издержки в долгосрочном периоде

Long Run / Долгосрочный период

English: In the long run, ALL inputs are variable. The firm can change plant size, technology, etc.

Русский: В долгосрочном периоде ВСЕ факторы переменные. Фирма может менять размер завода, технологию.

Returns to Scale / Отдача от масштаба

What happens to output when ALL inputs are increased by the same proportion?

Что происходит с выпуском, когда ВСЕ факторы растут в одной пропорции?

Type / ТипCondition / УсловиеExample / Пример
Increasing / ВозрастающаяOutput increases MORE than inputs (e.g., 2× inputs → 2.5× output)Specialization benefits / Специализация
Constant / ПостояннаяOutput increases SAME as inputs (2× inputs → 2× output)Many industries / Многие отрасли
Decreasing / УбывающаяOutput increases LESS than inputs (2× inputs → 1.5× output)Coordination costs / Издержки координации

Long Run Average Cost (LRAC) / Долгосрочные средние издержки

LRAC = the envelope of all short-run ATC curves. Each point = minimum cost for that output level with optimal plant size.

LRAC = огибающая краткосрочных ATC. Каждая точка = минимум издержек при оптимальном размере завода.

Economies and Diseconomies of Scale / Эффект масштаба

Term / ТерминMeaning / Значение
Economies of scaleLRAC falls as output increases / LRAC падает при росте Q
Constant returnsLRAC flat / LRAC горизонтальна
Diseconomies of scaleLRAC rises as output increases / LRAC растёт при росте Q

Sources of economies of scale: Bulk buying, specialization, spreading fixed costs, R&D.

Источники эффекта масштаба: Оптовые закупки, специализация, распределение постоянных издержек.

Industry Structure / Структура отрасли

Shape of LRAC affects market structure:

  • U-shaped with large MES → Few large firms (oligopoly) / Немного крупных фирм
  • Flat LRAC over wide range → Many firms (competitive) / Много фирм
