Cost Functions (Long Run) — Издержки в долгосрочном периоде

Long Run / Долгосрочный период

English: In the long run, ALL inputs are variable. The firm can change plant size, technology, etc.

Русский: В долгосрочном периоде ВСЕ факторы переменные. Фирма может менять размер завода, технологию.

Returns to Scale / Отдача от масштаба

Что происходит с выпуском, когда ВСЕ факторы растут в одной пропорции?

Industry Structure / Структура отрасли

Shape of LRAC affects market structure:

  • U-shaped with large MES → Few large firms (oligopoly) / Немного крупных фирм
  • Flat LRAC over wide range → Many firms (competitive) / Много фирм

Long Run Average Cost (LRAC): Detailed Graph / LRAC: детальный график

The LRAC curve is the envelope of all possible short-run ATC curves. Each point on LRAC represents the minimum average cost achievable at that output level, assuming the firm chooses the optimal plant size.

LRAC — огибающая всех возможных краткосрочных ATC. Каждая точка LRAC — минимум средних издержек при оптимальном размере завода.

Key points:

  • Each ATC curve represents a different plant size (fixed input level).
  • The LRAC curve touches each ATC at its minimum point (or at the point where that plant size is optimal).
  • To the left of the tangency: LRAC < ATC (firm should expand plant).
  • To the right of the tangency: LRAC < ATC (firm should contract plant).
  • At the tangency: LRAC = ATC (optimal plant size for that output).

Важно: Каждая ATC — свой размер завода. LRAC касается ATC в точке, где этот размер оптимален. Слева от касания: LRAC < ATC (нужно расширяться). Справа: LRAC < ATC (нужно сокращаться).

Minimum Efficient Scale (MES) / Минимальный эффективный масштаб

MES is the smallest output level at which LRAC reaches its minimum. Below MES, the firm has not fully exploited economies of scale. Above MES, diseconomies may set in (if LRAC rises).

MES — минимальный объём выпуска, при котором LRAC достигает минимума. Ниже MES фирма не использует весь эффект масштаба.

Long Run Total Cost (LRTC) and Long Run Marginal Cost (LRMC) / LRTC и LRMC

Long Run Total Cost (LRTC) / Долгосрочные общие издержки

LRTC = minimum total cost to produce each output level when all inputs are variable. LRTC = LRAC × Q. The LRTC curve shows how total cost changes in the long run.

LRTC — минимальные общие издержки при каждом объёме выпуска, когда все факторы переменные. LRTC = LRAC × Q.

Shape: LRTC typically rises at a decreasing rate (economies of scale), then at an increasing rate (diseconomies). The slope of LRTC = LRMC.

Форма: LRTC сначала растёт медленнее (эффект масштаба), затем быстрее (антиэффект). Наклон LRTC = LRMC.

Long Run Marginal Cost (LRMC) / Долгосрочные предельные издержки

LRMC = ΔLRTC / ΔQ = slope of LRTC. LRMC shows the extra cost of producing one more unit when the firm can adjust all inputs optimally.

LRMC = ΔLRTC / ΔQ = наклон LRTC. LRMC — дополнительные издержки на ещё одну единицу при оптимальной корректировке всех факторов.

Relationship: LRMC and LRAC

  • When LRMC < LRAC, LRAC is falling (economies of scale).
  • When LRMC > LRAC, LRAC is rising (diseconomies of scale).
  • LRMC crosses LRAC at the minimum of LRAC (at MES).

Когда LRMC < LRAC, LRAC падает (эффект масштаба). Когда LRMC > LRAC, LRAC растёт. LRMC пересекает LRAC в минимуме LRAC (в точке MES).

Summary:

  • LRAC is U-shaped: falls (economies of scale), then rises (diseconomies).
  • LRMC crosses LRAC at the minimum of LRAC (MES).
  • Below MES: LRMC < LRAC (economies of scale).
  • Above MES: LRMC > LRAC (diseconomies of scale).
  • At MES: LRMC = LRAC (constant returns to scale).

Итог: LRAC U-образная. LRMC пересекает LRAC в минимуме (MES). Ниже MES — эффект масштаба; выше — антиэффект.

Sources of Economies and Diseconomies of Scale / Источники эффекта и антиэффекта масштаба

Economies of Scale (LRAC falling) / Эффект масштаба

  • Specialization: Workers can specialize in specific tasks → higher productivity.
  • Bulk buying: Large firms get discounts on inputs.
  • Spreading fixed costs: R&D, advertising, overhead spread over more units.
  • Technology: Large-scale production may allow use of more efficient technology.
  • Financial: Easier access to capital, lower interest rates.

Специализация, оптовые закупки, распределение постоянных издержек, технологии, финансирование.

Diseconomies of Scale (LRAC rising) / Антиэффект масштаба

  • Coordination problems: Harder to manage large organizations.
  • Communication: Information may be lost or distorted in large hierarchies.
  • Bureaucracy: Red tape slows decision-making.
  • Motivation: Workers may feel less connected to the firm's goals.

Проблемы координации, коммуникации, бюрократия, мотивация.
