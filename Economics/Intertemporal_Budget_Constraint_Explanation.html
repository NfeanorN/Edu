Intertemporal Budget Constraint — Межвременное бюджетное ограничение

What is Intertemporal Budget Constraint? / Что такое межвременное бюджетное ограничение?

The intertemporal budget constraint extends the budget constraint to choices across time periods (e.g., today vs tomorrow, period 1 vs period 2). It shows how a consumer can allocate consumption between periods, taking into account interest rates and the ability to save or borrow.

Межвременное бюджетное ограничение расширяет обычное бюджетное ограничение на выбор между временными периодами (сегодня vs завтра, период 1 vs период 2). Показывает, как потребитель может распределить потребление между периодами с учётом процентной ставки и возможности сберегать или занимать.

Two-Period Model / Модель двух периодов

We consider two periods:

  • Period 1 (today / сегодня): income Y₁, consumption C₁
  • Period 2 (tomorrow / завтра): income Y₂, consumption C₂
  • Interest rate r (the rate at which you can save or borrow / ставка, по которой можно сберегать или занимать)

Рассматриваем два периода: период 1 (сегодня) и период 2 (завтра), с доходами Y₁, Y₂ и потреблением C₁, C₂ при процентной ставке r.

Saving and Borrowing / Сбережения и заимствование

If C₁ < Y₁, you save S = Y₁ − C₁. In period 2, you have Y₂ plus your savings with interest: Y₂ + S(1 + r) = Y₂ + (Y₁ − C₁)(1 + r).

Если C₁ < Y₁, вы сберегаете S = Y₁ − C₁. В периоде 2 у вас Y₂ плюс сбережения с процентами: Y₂ + S(1 + r).

If C₁ > Y₁, you borrow B = C₁ − Y₁. In period 2, you must repay Y₂ − B(1 + r) = Y₂ − (C₁ − Y₁)(1 + r) for consumption.

Если C₁ > Y₁, вы занимаете B = C₁ − Y₁. В периоде 2 вы должны вернуть B(1 + r), поэтому потребление C₂ = Y₂ − B(1 + r).

The Intertemporal Budget Constraint / Межвременное бюджетное ограничение

The constraint states that the present value of consumption equals the present value of income:

Ограничение: приведённая стоимость потребления равна приведённой стоимости дохода:

C₁ + C₂/(1 + r) = Y₁ + Y₂/(1 + r)

Present value of consumption = Present value of income

Приведённая стоимость потребления = Приведённая стоимость дохода

Present value (PV) converts future amounts to today's value. C₂/(1 + r) is the present value of period 2 consumption: if you want to consume C₂ in period 2, you need to save C₂/(1 + r) in period 1.

Приведённая стоимость (PV) переводит будущие суммы в сегодняшнюю стоимость. C₂/(1 + r) — приведённая стоимость потребления периода 2.

Alternative form / Альтернативная форма

Rearranging: C₂ = Y₂ + (Y₁ − C₁)(1 + r)

This says: period 2 consumption = period 2 income + (period 1 savings) × (1 + interest rate).

Перегруппировка: C₂ = Y₂ + (Y₁ − C₁)(1 + r). Потребление периода 2 = доход периода 2 + (сбережения периода 1) × (1 + ставка).

The Intertemporal Budget Line / Межвременная бюджетная линия

On a graph with C₁ on the horizontal axis and C₂ on the vertical axis, the budget line shows all combinations (C₁, C₂) that satisfy the constraint.

На графике с C₁ по оси X и C₂ по оси Y бюджетная линия показывает все комбинации (C₁, C₂), удовлетворяющие ограничению.

Бюджетная линия: наклон = −(1 + r). Точка E = (Y₁, Y₂) — endowment point (точка наделённости).

Intercepts / Пересечения

  • Horizontal intercept: If C₂ = 0, then C₁ = Y₁ + Y₂/(1 + r). Maximum period 1 consumption (spend all income today).
  • Vertical intercept: If C₁ = 0, then C₂ = Y₂ + Y₁(1 + r). Maximum period 2 consumption (save all period 1 income).

Горизонтальное пересечение: C₁ = Y₁ + Y₂/(1 + r) (максимальное потребление периода 1). Вертикальное: C₂ = Y₂ + Y₁(1 + r) (максимальное потребление периода 2).

Slope / Наклон

The slope of the budget line is −(1 + r). This represents the opportunity cost of period 1 consumption in terms of period 2 consumption: consuming 1 more unit in period 1 means giving up (1 + r) units in period 2.

Наклон бюджетной линии = −(1 + r). Это альтернативная стоимость потребления периода 1 в единицах потребления периода 2: потребление ещё одной единицы в периоде 1 означает отказ от (1 + r) единиц в периоде 2.

Endowment Point / Точка наделённости

The endowment point E = (Y₁, Y₂) is the consumption bundle if you neither save nor borrow (consume exactly your income in each period). It lies on the budget line.

Точка наделённости E = (Y₁, Y₂) — комбинация потребления без сбережений и займов (потребляете ровно свой доход в каждом периоде). Лежит на бюджетной линии.

  • Above E: C₂ > Y₂, C₁ < Y₁ → you save in period 1, consume more in period 2.
  • Below E: C₂ < Y₂, C₁ > Y₁ → you borrow in period 1, consume less in period 2 (repay debt).
  • At E: C₁ = Y₁, C₂ = Y₂ → no saving or borrowing.

Выше E: сберегаете в периоде 1. Ниже E: занимаете в периоде 1. На E: без сбережений и займов.

Example / Пример

Given: Y₁ = 100€, Y₂ = 50€, r = 0.1 (10%)

Дано: Y₁ = 100€, Y₂ = 50€, r = 0.1 (10%)

Budget constraint: C₁ + C₂/(1.1) = 100 + 50/(1.1) = 100 + 45.45 = 145.45€

Бюджетное ограничение: C₁ + C₂/1.1 = 145.45€

What Changes the Budget Line? / Что меняет бюджетную линию?

Change / ИзменениеEffect / Эффект
Interest rate r increases / Ставка r растётLine rotates around E, steeper slope / Линия поворачивается вокруг E, наклон круче
Interest rate r decreases / Ставка r падаетLine rotates around E, flatter slope / Линия поворачивается вокруг E, наклон положе
Y₁ increases / Y₁ растётLine shifts outward (parallel shift) / Линия сдвигается наружу
Y₂ increases / Y₂ растётLine shifts outward (parallel shift) / Линия сдвигается наружу

Link to Consumption Function / Связь с функцией потребления

In macroeconomics, the intertemporal budget constraint underlies the consumption function C = C(Y, Ye, wealth, r), where consumption depends on current income, expected future income, wealth, and the interest rate.

В макроэкономике межвременное бюджетное ограничение лежит в основе функции потребления C = C(Y, Ye, богатство, r), где потребление зависит от текущего дохода, ожидаемого будущего дохода, богатства и процентной ставки.
