Externalities — Внешние эффекты

What is an Externality? / Что такое внешний эффект?

English: A cost or benefit that affects a third party not involved in the transaction. Market fails to account for social cost/benefit.

Русский: Издержки или выгоды для третьей стороны, не участвующей в сделке. Рынок не учитывает общественные издержки/выгоды.

Type / ТипMeaning / ЗначениеExample / Пример
Negative externalityCost imposed on othersPollution from factory
Positive externalityBenefit to othersVaccination, education

Negative Externality / Отрицательный внешний эффект

Private cost < Social cost. Market produces TOO MUCH. Solution: tax (Pigouvian tax) to internalize the externality.

Частные издержки < Общественные. Рынок производит СЛИШКОМ МНОГО. Решение: налог.

P
  |
  |  MSC (Social cost)
  |   \
  |    \  * P_social  ← Optimal (tax brings here)
  |     \  |
  |      \ | MPC (Private cost)
  |       \|  \
  |        *---*--- D
  |        |   |
  |     Q_opt  Q_market  (market produces too much)
  |
  0+----+----+----+----+ Q

Pigouvian tax = size of externality per unit. Shifts MPC up to MSC.

Налог Пигу = величина внешнего эффекта на единицу.

Positive Externality / Положительный внешний эффект

Private benefit < Social benefit. Market produces TOO LITTLE. Solution: subsidy.

Частная выгода < Общественная. Рынок производит СЛИШКОМ МАЛО. Решение: субсидия.

P
  |
  |  MSB (Social benefit)
  |   /
  |  /  * P_opt  ← Subsidy raises Q
  | /   |
  |/    | S (MC)
  |     |
  |  *--*--- D (Private benefit)
  |  |
  |  Q_market  Q_opt
  |
  0+----+----+----+----+ Q
