# Аватар: Легенда об Аанге — Субтитры (React)

Клон проекта просмотра субтитров на React (Vite + TypeScript).

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Структура

- **/** — список сезонов и серий
- **/episode/1/5** — субтитры серии 5, сезон 1

Данные субтитров: `src/data/s01e05.ts`. Чтобы добавить полный текст с сайта, скопируйте реплики в массив `entries`.
